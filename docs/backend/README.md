# Red Sea Express Backend

Backend API for Red Sea Express Logistics Platform.

## Technology
- Python
- FastAPI
- PostgreSQL
- Docker

## Features
- User management
- Vendor onboarding
- Trade Hub services
- Import requests
- Export offers
- Logistics requests
- Wallet system
- AI Assistant integration

## Setup

1. Create environment file:
```

cp .env.example .env

```

2. Install requirements:
```

pip install -r requirements.txt

```

3. Run backend:
```

python main.py

```

## API
Backend runs on: `http://localhost:8000`

## Main Modules
- Authentication
- Users
- Vendors
- Trade Hub
- Shipping
- Containers
- Wallet
- AI Assistant (Aisha)
